HI-CURV ATOMIC PARAMETER MODEL — “82”
============================================
“82” = the complete set of atomic-level parametric sliders that control all visual, spatial, motion, density, interaction, and expressive behavior across the entire HI-CURV system.
These sliders exist ONLY at the atomic level.
Molecules never define sliders — they only route slider values into atoms.
Screens never define code — they only supply values to the sliders.
Every visual change in the system comes from some combination of these 82 knobs.
This is the entire parametric surface of your OS.

================================
HOW THEY ARE USED
================================
Atoms expose sliders → Molecules configure sliders → Screens choose values.
An atom is a bag of sliders.
A molecule chooses which atoms to activate and which slider values to pass into them.
A screen simply supplies numeric values (0–1 or presets like S/M/L mapped into numeric ranges).
This is why your system can morph any button, any card, any calendar, any Kanban, any organ — instantly — using the exact same engine.
