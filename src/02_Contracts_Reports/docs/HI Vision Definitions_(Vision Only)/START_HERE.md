# Definitions — Start Here

**AI and humans:** Read this folder before `docs/SYSTEM_MASTER`.  
This folder explains the system **conceptually**, not technically. For pipelines, contracts, and code locations, use `docs/SYSTEM_MASTER/`.

---

## The 5 system groups

1. **Core System** — Brain (content engine), Organs (view engines), Assistant (voluntary/involuntary actions), System 7 Media, Features, and Content Views.

2. **UI System** — Atoms, Molecules, Compounds, JSON-driven UI, and UX Media primitives (Surface, Media, Content).

3. **Morph System** — 0→1 system, sliders, parametric styling, density, and motion.

4. **Action System** — Behavior engine, verbs, triggers, sequences, and the 6×7 grid.

5. **Guide System** — Core philosophy, one-screen model, UI/UX rules, analogies, and GPT constraints.
